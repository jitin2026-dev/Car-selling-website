## 📌 Overview

The **Car Selling Website** is a responsive front-end project designed to showcase different cars in a modern and attractive layout. The website is built using only **HTML5** and **CSS3**, focusing on clean design, responsiveness, and a great user experience.

---

## ✨ Features

- 🚘 Attractive Landing Page
- 📱 Fully Responsive Design
- 🎨 Modern User Interface
- 🖼️ Car Showcase Section
- 📞 Contact Section
- ⚡ Fast Loading
- 💻 Clean Code Structure

---

| Technology | Purpose |
|------------|---------|
| HTML5 | Website Structure |
| CSS3 | Styling & Responsive Design |
